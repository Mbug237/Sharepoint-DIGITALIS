SHAREPOINT

Ce sharepoint permet particulièrement de stocker des documents d’entreprise pour faciliter l’utilisation de ces derniers. Il est constitué de deux dossiers à savoir frontshare (frontend) et sharepoint (backend). Lors du lancement de l’application il présente plusieurs sections à savoir :

    • RH
    • Directeur technique
    • Dir com
    • Administrateur
    • Documents 

La section qui nous intéresse est Documents car c’est ici qu’on devra enregistre des documents afin de les réutiliser en entreprise.

Fonctionnalités :

    • Ajouter /supprimer une catégorie
    • Ajouter un document avec les options suivantes (lire, modifier, télécharger, supprimer)

Comment ça marche ?

Une fois sur le dashboard cliquer sur documents. Vous serez dirigé sur la page d’ajout de catégories, Une fois une catégorie ajoutée, vous pouvez cliquer sur celle ci pour voir les documents qu’elle possède ou alors supprimer la catégorie. Une fois dans une catégorie vous pouvez ajouter un document et une fois enregistré, vous pouvez voir le document enregistré avec les options lire, modifier, télécharger et supprimer.
